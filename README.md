# 📚 Node.js & API REST — Formation 3 Jours

Bienvenue dans cette formation intensive sur **Node.js et les API REST**.

---

## 🎯 Objectifs

- Comprendre et utiliser Node.js côté serveur
- Construire des API REST avec Express.js
- Structurer un projet professionnel (architecture modulaire)
- Persister des données avec SQLite
- Sécuriser une API avec JWT et les bonnes pratiques

## 🚀 Pour les apprenants — Comment démarrer

### 1. Forker ce dépôt

Cliquez sur le bouton **Fork** en haut à droite de cette page.

### 2. Cloner votre fork

```bash
git clone https://github.com/VOTRE-PSEUDO/node-api-course.git
cd node-api-course
```

### 3. Structure de rendu

Pour chaque évaluation, créez votre code dans le dossier correspondant :

```
evaluations/
├── jour1/     ← Votre code pour l'évaluation du jour 1
├── jour2/     ← Votre code pour l'évaluation du jour 2
└── jour3/     ← Votre code pour l'évaluation du jour 3
```

### 4. Envoyer votre rendu

Une fois votre travail pushé, envoyez le lien vers votre fork au formateur :

```
https://github.com/VOTRE-PSEUDO/node-api-course
```
